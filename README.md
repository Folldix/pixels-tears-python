# Pixels and Tears — PyGame порт

Це порт проєкту з Godot Engine на **Python + PyGame** (папка `assets/` містить перенесені ресурси).

## Запуск

1. Встановити залежності:

```bash
python -m pip install -r requirements.txt
```

2. Запустити гру:

```bash
python main.py
```

## Керування

- **Меню**
  - **M** — обрати `male`
  - **F** — обрати `female`
  - **Enter** — старт локальної гри
  - **Esc** — вихід

- **Гра**
  - **WASD / Стрілки** — рух
  - **E** — взаємодія (ховає найближчий “предмет” в радіусі ~40 px)
  - **Esc** — повернутись у меню

## Відповідність Godot-логіці (поточний стан)

- **Гравець**: рух, анімації напрямків, вибір скіна, звук ходьби
- **Ворог**: з’являється через 5 хвилин і переслідує гравця (спрощено)

Мережевий режим (WebSocket) і пауза будуть додані наступним кроком.
